# HTML 基本命令

<h> 我是级标题 </h>:<h></h>=head 的缩写 分级的意义：为了使网页更有语义化 分为 6 个等级

<.h1> 我是 1 级标题 <./h1>

<.h2> 我是 2 级标题 <./h2>

<.h3> 我是 3 级标题 <./h3>

<.h4> 我是 4 级标题 <./h4>: 会独占一行

<.h5> 我是 5 级标题 <./h5>

<.h6> 我是 6 级标题 <./h6> 没有。才能生成标题 这里只是因为显示问题加入

写代码之前需要先保存一下

段落标签 <P></P>

换行标签 <br/>: 是 break 的缩写意思是 打断 。换行

加粗标签 <strong> </strong>: 加粗非标题文字的方法

<b> </b> 同上方一样也是加粗标签 推荐优先使用 strong 加粗方法

倾斜标签 <em> </em>. <i></i > 两个标签都是可以变倾斜的推荐优先使用 em 倾斜方法

删除线标签 <del></del> <S></S > 两个都是删除线标签推荐优先使用 del

下划线命令 <ins></ins> <u></u > 两个都是下滑线命令推荐优先使用 ins

盒子标签 <div></div> <span></span>

div 是 division 的缩写 表示分割划区的意思一行只能有一个 div

span 可以理解为跨距并不会独占一行

div 可以理解为大盒子 span 可以理解为小盒子

图片标签 <img> 需要代码命令结合使用 <.img src="路径或链接"> 没有里面的点才可以

<.img src="路径或链接" title> 加 title 可以在点击图片时出现文字 这里也是

<img src="路径或链接"alt = 图片加载失败 /> 加 alt 可以在加载图片失败的时候显示文字

<img src="路径或链接"width=""> width: 给设置图像的宽度

<img src="路径或链接"height=""> height: 给图像设置高度

<img src="路径或链接"border=""> borde: 设置图像边框

/ 图像位于下一级路径需要添加的符号 列示 D 盘 / 新建文件夹 / 图片 1

../ 图像位于上一级路径所需符号 列示 D 盘 / 图片 / 文件夹 使用方式 D 盘 ../ 图片 /

<a> </a> : a 是超链接的意思 跳转出去 加入属性词 href 如 <a href:"链接" 跳转过去地方名字 > </a>

blank: 加如这个命令词 可以在原有网页保留情况下打开新网页 不加则原来网页点击后原地跳转网页
使用写作 <a href:"链接" 跳转过去地方名字 "target="_blank"> 名字 </a>

#号 如果页面没有计划或者没有做完可以使用＃号 #代表空链接使用方法如下
<a href:"#" 跳转过去地方名字> </a>

点击图片跳转格式 </a href="链接地址"> </img src="路径或链接"> </a>

同意页面内跳转命令 <a h0ref="# 自命名"> 跳转处名字 <h30 id="自命名"> 被跳转处名字 </h3> 这里代码没有 0 为了展示加的

ctrl+/ 可以直接创建 <!-- --> 命令：注释标签 可以让注释不显示到网址上

&nbsp 空格命令 一个词站一个空格 列示 & nbsp；&nbsp；&nbsp 就是三个空格注意要用分号隔开

4&l1t; &1gt; 是大于号 和小于号 不能直接输入 < > 这里的符号没有数字为了展示才加
